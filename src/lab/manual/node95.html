Self Tuning Controller

Figure 9.13: Xcos diagram for self tuning controller

The parameters of the controller are determined dynamically using the temperature values for every sampling time. For this, the formulae derived in section 9.4.3 are used. The formulae for the control effort are same as the conventional PI and PID controllers. So the PI/PID settings are calculated for every sampling time and the control effort is calculated thereafter using the formulae derived for conventional controllers.
